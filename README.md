To get started with launching the server, follow these steps:

1. **Clone the Repository:**
    ```bash
    git clone https://github.com/vetal-hovenko/wordle-back.git
    ```

2. **Install Dependencies:**
    ```bash
    npm install
    ```
3. **Go to wordle-back:**
    ```bash
    cd ./wordle-back
    ```
    
4. **Run the Server:**
    ```bash
    node server.js
    ```
